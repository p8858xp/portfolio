# Homework 05

## Directory Structure

You finished application should look similar to the following (note, some of these files, such as `src/app.js` and `.eslintrc.js`, may have already been created for you):

in `/` (project root)...

* `package.json`
* `node_modules`
* `.gitignore`
* `.eslintrc.js`
* `/src`
	* `app.js`
	* `public`
	* `img`
	* `css`
		* your css file
	* `views`
		* `layout.hbs`
		* any other views
